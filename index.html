Test CSS on any page without touching source code.

CSS Scratchpad is a lightweight browser tool for temporarily applying custom CSS to the current webpage. It is built for development, testing, visual debugging, and quick design experiments.

Private by design.

CSS Scratchpad does not need accounts, servers, analytics, tracking, or personal data. The CSS you write stays local in your browser.

  • No tracking
  • No analytics
  • No remote servers
  • No personal data collection
  • CSS is applied only when you click Apply
